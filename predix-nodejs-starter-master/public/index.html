• Welcome to a simple Node.js Express App

  • What's Inside

  • This app displays index.html (that's the page you are reading now). Notice it's in the public directory?
  • Node.js loads the app on the server side in the file app.js. Notice this file is in the server directory?
  • The routes/index.js is used to set up a route url for this app. Routes are how Node.js matches logic tied to a url. Notice this file is in the server/routes directory?
  • The other files become enabled in future tutorials.
